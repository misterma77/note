添加路由与子路由

children 属性

一个路由中的 children 表示嵌套的子路由

  • 子路由的 path 如果以 / 开头表示根路径，不再基于父级路径，否则基于父级 path
  • 如果一个子路由的 path 为空，表示为默认子路由
  • 如果一个路由有默认子路由，则父级 name 属性需要设置给这个默认子路由

数组

数组时存储1个或多个数据的容器，是一组内存空间
通常用来批量处理数据
这组内存空间的名字叫做数组名
数组对自身储存的数组元素没有限制，不限数量类型
通过数组下标(从0开始)来管理和操作数组
数组名.length 代表了数组的长度，储存元素的个数
因为下标从0开始，所以数组长度-1表示最后一个元素的下标

创建数组

传统写法：
  var arr = new Array();
  var str = new String();
字面量的写法：
  var arr = [1,2,3,4];
